# BABYDOOM
## Clone of DOOM using the JavaScript Babylon.js engine.

[Azure hosting of the game](http://babydoom.azurewebsites.net/)

Current features:

+ Random enemy placement
+ Sprite animation
+ Random "world" generation (wall placement)
+ Gun shot
+ Bullet collisions (buggy though)
